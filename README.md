Moc-Cmus_Config
===============

The configuration about MOCP

##Usage
- Copy the .moc folder to $HOME
    <pre><code>$ cp -r .moc ~/
    </code></pre>
- If show error: `Configuration file is not secure: /home/marslo/.moc/config`, then set the file mode of config as 755:
    <pre><code>$ chmod 755 ~/.moc/config
    </code></pre>
